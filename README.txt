SETUP GUIDE

1. Open config.js
2. Paste your Discord Bot Token

START COMMAND:
npm start

RENDER:
- Build Command:
npm install

- Start Command:
npm start

BOT INTENTS:
Enable all Privileged Gateway Intents in Discord Developer Portal.

COMMANDS:

$play song
$skip
$stop
$pause
$resume
$queue

$voice mute @user
$voice unmute @user
$voice muteall
$voice unmuteall

$voice deafen @user
$voice undeafen @user
$voice deafenall
$voice undeafenall

$voice kick @user
$voice kickall

$voice move @user #vc
$voice moveall #vc
